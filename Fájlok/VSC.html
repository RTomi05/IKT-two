Mik is azok a Code Editorok?




A kódszerkesztő olyan egyszerű szövegszerkesztő környezet, illetve program, amit kifejezetten szoftverfejlesztők
általi használatra és a programok forráskódjának szerkesztésére hoztak létre.

A kódszerkesztők nagymértékben hasonlítanak az egyszerűbb általános célú szövegszerkesztő programokhoz, de azoktól jelentősen eltérő funkciókat kínálnak. Így pl. nem teszik lehetővé a szöveg formázását, ugyanakkor rendelkeznek a szintaxiskiemelés, az automatikus behúzás vagy az automatikus kódkiegészítés képességével.

Ezen eszközök általában még több funkcióval rendelkezhetnek, például a szerkesztett program fordításához, néha még debuggolásához is, de összetett refaktorációs képességekkel vagy UI-szerkesztővel nem rendelkeznek. Ebben jelentős mértékben eltérnek az Integrált Fejlesztői Környezetektől (IDE), ami számos egyéb, a kódszerkesztéssel össze nem függő, illetve rendkívüli módon összetett funkciót is tartalmaznak.


A kódszerkesztők nagy előnye az IDE -kkel szemben, hogy a kevesebb funkció miatt sokkal gyorsabban töltődnek be és működnek, illetve a menüjeik is egyszerűbbek. Ezzel összefüggésben a legtöbb kódszerkesztő program teljes mértékben ingyenes is.


A legismertebb kódszerkesztő programok:

  • Atom
  • Notepad++
  • Sublime
  • Visual Studio Code


Debug: programhibákat kiküszöböl vagy ellenőriz.

IDE: integrált programozási/fejlesztői környezet.
